= Руководство по ArchiMate

Это руководство предоставляет объяснение и обзор стандарта ArchiMate версии 3.2, предназначенное для помощи архитекторам и специалистам в моделировании ИТ-архитектуры.

WARNING: Это руководство содержит *переформулированное* объяснение концепций ArchiMate и *не является* официальной спецификацией. Для точного определения элементов и связей обращайтесь к официальной спецификации от The Open Group.
